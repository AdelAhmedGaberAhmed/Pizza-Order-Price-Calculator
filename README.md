# 🍕 Pizza Order System - Price Calculator

## 📌 Overview
This is a simple **Windows Forms application** built using C# that simulates a pizza ordering system.

The main idea of the project is to dynamically update the total price based on user-selected ingredients (add-ons or removed items).

---

## 🛠️ Technologies Used
- C#
- Windows Forms (.NET Framework)
- Object-Oriented Programming (OOP)
- Event-driven programming

---

## 🍕 Project Features
- Base pizza selection
- Add/remove toppings dynamically
- Real-time price calculation
- Instant UI update when changes occur
- Simple and user-friendly interface

---

## ⚙️ How It Works
- Each topping has a fixed price value
- When the user selects a topping, its price is added to the total
- When the user removes a topping, its price is deducted
- Final price is always updated instantly on the screen
